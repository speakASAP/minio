## MinIO
S3-compatible object storage server used by this service.

## Bucket
Private logical object container. Current docs identify speakasap-records as the configured recordings bucket while records appears as the bucket concept.

## Object Key
Path-like object identifier. Lesson recordings use YYYY/MM/DD/lesson_<uuid>.mp3.

## Presigned URL
Temporary S3 URL generated by an authorized service for read access without making the bucket public.

## SigV4
AWS Signature Version 4 signing protocol. Proxy routing must preserve host, path, and authorization semantics.

## Consumer
Trusted application that writes or reads objects through S3 APIs, currently speakasap-portal and runlayer.
